Project Description:
This project is a simple implementation of network scanning tools using Python.
It includes three basic tools:
1. Ping Scanner
2. ARP Scanner
3. Nmap Scanner

These tools help to check network connectivity, identify devices in the network,
and scan open ports of a target system.

Tools Used:
- Python
- Nmap
- Visual Studio Code

Files in this Project:
1. ping_scanner.py  - Checks whether a host is reachable using ping command.
2. arp_scanner.py   - Displays the ARP table and shows devices in the network.
3. nmap_scanner.py  - Performs basic network scanning using Nmap.
4. screenshots.docx - Contains output screenshots of the programs.
5. README.txt       - Project description and instructions.

How to Run the Programs:

1. Open the project folder in Visual Studio Code.
2. Open the terminal.
3. Run the Python files using the following commands:

python ping_scanner.py
python arp_scanner.py
python nmap_scanner.py

Requirements:
- Python installed
- Nmap installed
- Internet or network connection

Conclusion:
This project demonstrates basic network scanning techniques using Python and
system commands. It helps understand how tools like Ping, ARP and Nmap work.
